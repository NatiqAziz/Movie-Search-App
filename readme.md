# 🎬 Movie Search App

**Search movies. Get instant details.**

## Live Demo
link: https://ombd-movie-search-app.netlify.app/

## Features

🔍 Search by title  
🎭 View posters, ratings, cast  
📱 Fully responsive  

## Tech

HTML • CSS • JavaScript • OMDB API

## Setup

Get API key from: https://www.omdbapi.com/apikey.aspx  
Replace `YOUR_API_KEY_HERE` in `app.js` with your key.
